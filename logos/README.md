# CivicPress Logo Usage Guide

This directory contains the official CivicPress logos and icons. These assets may be used in municipal documents, websites, reports, presentations, and media coverage when referring to CivicPress.

The goal of this guide is to ensure correct and consistent use of the CivicPress identity.

## Available Formats

### SVG  
Recommended for web, software, and high-resolution displays.

### PNG  
Recommended for office documents, slide decks, and situations where SVG is not supported.

### Variants  
- Black versions for light backgrounds  
- White versions for dark backgrounds  
- Icons for square or compact use  
- Full logo for general use  

## How to Use the Logo

### Minimum clear space  
Leave enough space around the logo so it is not crowded by text or other elements. A space equal to the height of the letter C in CivicPress is a safe minimum.

### Backgrounds  
Use the version that ensures maximum legibility.

- Use the black logo on white or light backgrounds.  
- Use the white logo on black or dark backgrounds.  
- Avoid placing the logo on photos or busy textures unless the contrast is strong and clear.

### Do not modify the logo  
To preserve the integrity of CivicPress' public identity, please avoid:

- Stretching or distorting the logo  
- Changing the colors  
- Adding outlines, drop shadows, or effects  
- Altering proportions between the icon and the CivicPress wordmark  
- Rotating or rearranging elements  

## File Naming

The directory is organized as follows:

- `icon-*.{svg,png}` Square icon version  
- `logo-*.{svg,png}` Full CivicPress logo  
- `*-black` Black version for light backgrounds  
- `*-white` White version for dark backgrounds
