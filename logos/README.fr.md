# Guide d'utilisation du logo CivicPress

Ce répertoire contient les logos et icônes officiels de CivicPress. Ces éléments peuvent être utilisés dans des documents municipaux, des sites web, des rapports, des présentations et des contenus médiatiques lorsqu'ils font référence à CivicPress.

L'objectif de ce guide est d'assurer une utilisation correcte et cohérente de l'identité visuelle de CivicPress.

## Formats disponibles

### SVG  
Recommandé pour le web, les logiciels et les affichages haute résolution.

### PNG  
Recommandé pour les documents bureautiques, les présentations et les contextes où le SVG n'est pas pris en charge.

### Variantes  
- Versions noires pour fonds clairs  
- Versions blanches pour fonds foncés  
- Icônes pour un usage carré ou compact  
- Logo complet pour l'usage général  

## Bonnes pratiques d'utilisation

### Espace minimal  
Laissez suffisamment d'espace autour du logo pour éviter qu'il soit collé à du texte ou à d'autres éléments. Un espace équivalant à la hauteur de la lettre C de CivicPress est un minimum recommandé.

### Arrière-plans  
Choisissez la version assurant la meilleure lisibilité.

- Utilisez le logo noir sur un fond blanc ou clair  
- Utilisez le logo blanc sur un fond noir ou foncé  
- Évitez de placer le logo sur des photos ou fonds complexes, sauf si le contraste est suffisant  

### Modifications interdites  
Pour préserver l'intégrité de l'identité de CivicPress, évitez:

- D'étirer ou déformer le logo  
- De changer les couleurs  
- D'ajouter des ombres ou effets  
- De modifier les proportions entre l'icône et le mot CivicPress  
- De faire pivoter ou réarranger les éléments  

## Nommage des fichiers

Le répertoire est organisé comme suit:

- `icon-*.{svg,png}` Icône carrée  
- `logo-*.{svg,png}` Logo complet  
- `*-black` Version noire pour fonds clairs  
- `*-white` Version blanche pour fonds foncés  

## Utilisation et droits

Le nom et le logo CivicPress sont des marques du projet CivicPress.  
Ils peuvent être utilisés librement pour identifier ou référencer CivicPress, à condition que:

- L'utilisation ne soit pas trompeuse  
- Aucun soutien officiel ne soit suggéré sans autorisation  
- Le logo ne soit pas modifié au-delà des règles décrites ci-dessus
